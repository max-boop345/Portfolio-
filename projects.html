Dijkstra's Algorithm Visualizer

This interactive tool allows you to visualize Dijkstra's shortest path algorithm in action. Add nodes and edges to the graph, then run the algorithm to see how it finds the optimal path.

  • Click anywhere to add a node.
  • Click on a node, then click another node to add an edge.
  • Click "Run Dijkstra" to start the algorithm.
  • Watch as the algorithm explores the graph and finds the shortest path!

Time Complexity: O(E + V log V)

Space Complexity: O(V)

Ant Colony Optimization

This simulation demonstrates Ant Colony Optimization, a metaheuristic algorithm inspired by the foraging behavior of ants. Ants deposit pheromones as they move, and these pheromones evaporate over time. The shortest path emerges as the most traveled route.

  • Blue ants are going to the food.
  • Red ants are returning to the nest with food.
  • Green trails represent pheromone concentrations.
  • Obstacles (dark circles) force ants to find alternative paths.

Inspired by: Mines Informatiques 2025 subject

Algorithm: Pheromone-based pathfinding
