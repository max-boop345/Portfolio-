My Projects

Here are some of the projects I've worked on, ranging from numerical simulations to algorithmic visualizations.

🎳

Bowling Ball Trajectory Simulation

Python NumPy Matplotlib

Numerical simulation of a bowling ball’s motion, accounting for friction, spin, and lane interactions. The equations of motion were solved using Euler integration, and the results were visualized to analyze the ball's trajectory under different initial conditions.

🌕

Lunar Space Elevator Simulation

Python SciPy Matplotlib

Modeled the dynamics of a space elevator on the Moon, considering cable tension, centrifugal force, and gravitational pull. The simulation compared the feasibility of a lunar elevator with Earth-based designs, highlighting the reduced material constraints due to the Moon's lower gravity.

🚀

Rocket Trajectory in Earth-Moon System

Python NumPy Matplotlib

Simulated a rocket’s orbit in the Earth-Moon system with delta-V maneuvers to adjust the trajectory. The simulation used n-body problem approximations to model the gravitational interactions between the rocket, Earth, and Moon.

🗺️

Dijkstra's Algorithm Visualizer

JavaScript HTML5 Canvas Algorithms

An interactive tool to visualize Dijkstra’s shortest path algorithm. Users can add nodes and edges, set weights, and watch the algorithm find the optimal path step-by-step. Demonstrates the O(E + V log V) time complexity in action.

🐜

Ant Colony Optimization

JavaScript p5.js Algorithms

Simulation of ants finding the shortest path between food and nest using pheromone trails. Inspired by the Mines Informatiques 2025 subject, this project demonstrates emergent behavior from simple rules: ants deposit pheromones, which evaporate over time, leading to the discovery of optimal paths.
